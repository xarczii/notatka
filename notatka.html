SVG
Skalowalna Grafika Wektorowa

SVG to format grafiki wektorowej do tworzenia dwuwymiarowych obrazów. Przykład składni SVG:

<svg width="100" height="100" xmlns="http://www.w3.org/2000/svg">
  <circle cx="50" cy="50" r="40" stroke="black" stroke-width="3" fill="red" />
</svg>

SVG pozwala na manipulowanie grafiką za pomocą stylów CSS i skryptów JavaScript, co czyni go bardzo elastycznym narzędziem do pracy z grafiką na stronach internetowych.

Wybrane atrybuty SVG:

  • width, height - określenie wymiarów rysunku.
  • fill - kolor wypełnienia elementu.
  • stroke, stroke-width - kolor i grubość obramowania.
  • cx, cy, r - współrzędne środka i promień okręgu.
  • viewBox - definiuje obszar widoczny grafiki.
HTML+CSS
HTML i CSS

HTML definiuje strukturę strony internetowej, a CSS odpowiada za jej wygląd. Przykład:

<!DOCTYPE html>
<html>
  <head>
    <style>
      body { background-color: lightblue; }
      h1 { color: navy; font-size: 24px; }
    </style>
  </head>
  <body>
    <h1>Witaj, Świecie!</h1>
  </body>
</html>

HTML wykorzystuje tagi do tworzenia elementów, takich jak nagłówki, akapity czy obrazy, a CSS umożliwia ich stylizację w sposób estetyczny i czytelny.

Wybrane atrybuty HTML:

  • id - unikalny identyfikator elementu.
  • class - przypisanie klasy CSS do elementu.
  • src, alt - adres obrazu i tekst alternatywny.
  • href, target - adres odnośnika i sposób jego otwarcia.
  • style - inline style dla elementu.

Wybrane atrybuty CSS:

  • color, background-color - kolory tekstu i tła.
  • margin, padding - odstępy wokół elementu.
  • font-family, font-size - czcionka i rozmiar tekstu.
  • border, box-shadow - obramowanie i cień elementu.
Canvas
Canvas

Canvas to element HTML5 umożliwiający rysowanie grafik za pomocą JavaScript. Przykład:

<canvas id="myCanvas" width="200" height="100" style="border:1px solid #000000;"></canvas>
<script>
  var c = document.getElementById("myCanvas");
  var ctx = c.getContext("2d");
  ctx.fillStyle = "red";
  ctx.fillRect(20, 20, 150, 75);
</script>

Canvas pozwala na tworzenie dynamicznych grafik, takich jak wykresy, animacje i gry, oferując pełną kontrolę nad każdym pikselem obrazu.

Wybrane atrybuty Canvas:

  • width, height - szerokość i wysokość obszaru rysowania.
  • fillStyle - kolor wypełnienia.
  • strokeStyle - kolor obramowania.
  • lineWidth - grubość linii.
  • font, textAlign - styl tekstu i wyrównanie.
Podstawy JavaScript
Przykłady w JavaScript

JavaScript pozwala na dodawanie interaktywności do stron internetowych. Poniżej znajdziesz kilka przykładów zastosowania:

Zegar
<script>
  function showTime() {
    const date = new Date();
    document.getElementById("clock").innerText = date.toLocaleTimeString();
    setTimeout(showTime, 1000);
  }
  window.onload = showTime;
</script>
<div id="clock" style="font-size: 24px; font-weight: bold;"></div>
Operatory matematyczne
<script>
  const a = 10, b = 5;
  console.log("Dodawanie: ", a + b);
  console.log("Odejmowanie: ", a - b);
  console.log("Mnożenie: ", a * b);
  console.log("Dzielenie: ", a / b);
</script>
Slider
<input type="range" id="slider" min="0" max="100" oninput="updateValue(this.value)">
<span id="sliderValue">50</span>
<script>
  function updateValue(val) {
    document.getElementById("sliderValue").innerText = val;
  }
</script>
Progress Bar
<progress id="progressBar" value="50" max="100"></progress>
<button onclick="increaseProgress()">Zwiększ</button>
<script>
  function increaseProgress() {
    const bar = document.getElementById("progressBar");
    if (bar.value < bar.max) {
      bar.value += 10;
    }
  }
</script>
addEventListener

Funkcja addEventListener pozwala przypisać zdarzenia do elementów w sposób dynamiczny:

<button id="myButton">Kliknij mnie</button>
<script>
  document.getElementById("myButton").addEventListener("click", function() {
    alert("Przycisk został kliknięty!");
  });
</script>
Popover w HTML

Popover to interaktywny element Bootstrap do wyświetlania treści po najechaniu lub kliknięciu:

<button type="button" class="btn btn-secondary" data-bs-toggle="popover" data-bs-content="Treść popovera">Pokaż Popover</button>
<script>
  var popoverTriggerList = [].slice.call(document.querySelectorAll('[data-bs-toggle="popover"]'));
  var popoverList = popoverTriggerList.map(function (popoverTriggerEl) {
    return new bootstrap.Popover(popoverTriggerEl);
  });
</script>
Podstawy PHP
Przykłady w PHP

PHP to język skryptowy do tworzenia dynamicznych stron WWW. Przykład:

<?php
  $servername = "localhost";
  $username = "root";
  $password = "";
  $dbname = "moja_baza";

  $conn = new mysqli($servername, $username, $password, $dbname);

  if ($conn->connect_error) {
    die("Błąd połączenia: " . $conn->connect_error);
  }

  $sql = "SELECT id, nazwa FROM produkty";
  $result = $conn->query($sql);

  if ($result->num_rows > 0) {
    while($row = $result->fetch_assoc()) {
      echo "ID: " . $row["id"]. " - Nazwa: " . $row["nazwa"]. "<br>";
    }
  } else {
    echo "Brak wyników";
  }

  $conn->close();
?>
